Recurso docente interactivo

Modelo computacional simple del control motor

Recurso conceptual e interactivo para docencia en neurorehabilitación. Integra control cognitivo, automatización, emoción, estado corporal y retroalimentación sensorial en un modelo simple, enseñable y escalable.

Arquitectura Flujo Variables Reglas Simulador

1. Arquitectura funcional del modelo

Entrada

Inputs del sistema

Entorno, objetivo motor, calidad sensorial, fatiga, dolor, estrés, atención disponible y aprendizaje previo.

Procesamiento

Módulos centrales

Corteza, ganglios basales, cerebelo, sistema límbico, tronco encefálico y médula espinal operan como red integrada.

Salida

Conducta observable

La red genera estabilidad, latencia de respuesta, variabilidad, eficiencia motora y riesgo motor.

Dominios del control

DominioFunciónEjemplo
CognitivoPlanificación, atención, decisión conscienteAprender una tarea nueva
AutomáticoControl rápido, eficiente y posturalCaminar sin pensar
EmocionalModulación por amenaza, miedo, motivaciónRigidez o torpeza bajo presión

Idea central

El movimiento no es producido por una sola estructura. Es el resultado de una interacción dinámica entre intención, automatización, emoción, estado corporal y retroalimentación.

Regla pedagógica útil: cuando baja la automatización o la excitabilidad espinal, el sistema depende más de control cortical. Eso aumenta el costo cognitivo, la variabilidad y el riesgo de error.

3. Variables del modelo

Entradas

VariableSignificancia
FatigaReduce automatización y excitabilidad espinal.
DolorInterfiere con reclutamiento, precisión y confianza motora.
EstrésAumenta carga emocional e inestabilidad organizativa.
Atención disponibleDetermina cuánto control consciente puede sostenerse.
Aprendizaje previoFavorece automatización y eficiencia.
Calidad sensorialDefine cuánta información fiable recibe el sistema.
Demanda de tareaExige mayor ajuste si la tarea es compleja.
Tarea dualCompite por recursos cognitivos.

Salidas

SalidaInterpretación
EstabilidadCapacidad global de mantener control funcional.
Latencia de respuestaRapidez para corregir perturbaciones.
Variabilidad del movimientoConsistencia o dispersión de la respuesta.
Eficiencia motoraRelación entre control logrado y costo del sistema.
Riesgo motorProbabilidad de error, pérdida de control o lesión.

4. Reglas causales simples

Reglas del sistema

  • ↑ fatiga → ↓ excitabilidad espinal
  • ↑ fatiga → ↓ automatización postural
  • ↑ estrés → ↑ carga emocional
  • ↓ calidad sensorial → ↑ error y variabilidad
  • ↑ aprendizaje previo → ↓ necesidad de control consciente
  • ↑ tarea dual → ↓ atención disponible efectiva

Fórmulas conceptuales

automatización efectiva = (automatización postural + aprendizaje previo + ajuste cerebeloso) - (fatiga + dolor + estrés)
dependencia cortical = demanda de tarea + tarea dual + error predicho + baja automatización
riesgo motor = (latencia + variabilidad + dependencia cortical) - estabilidad
